This page holds the high level outline for the template library. Each section groups related files. Each row is a single planned template with a short description and a status tag. When a template moves from idea to in use, only this page needs to be updated to keep the whole track in view.

• Change the status pill text and class when work begins or ships.
• Add notes like links or dates in the notes column.
• Duplicate whole sections when new sub projects appear.

Page layout templates

Full page skeletons to reuse across Vigil. Each file is a starting point for new public pages.

Set 01
HomeLayout.html
Home layout with hero header, two or three featured cases, an intro block, a current focus strip, newsletter slot, and a compact “how Vigil works” section.
Todo Base for homepage rebuild.
ArticleLayout.html
Long form article layout with title, dek, meta row, scrollable body, inline pull quotes, and space for references or footnotes.
Todo
CaseDossierLayout.html
Dossier layout for a single case. Status bar for open, stalled, archived, a side timeline column, key facts grid, and evidence links block.
Todo Important for first case launch.
SeriesIndex.html
Index page for a series or theme for example Signals or Quiet Ledgers. Header, filter tabs or chips, and a grid or list of cards for each entry.
Todo
LibraryArchive.html
Chronological and filterable archive of all pieces. Search bar, topic and region and type filters, and list layout with concise meta rows.
Todo
AboutLayout.html
About layout for future edits and variants. Holds mission, methodology, notes, and any reference sections while staying consistent with current About.
In progress Current About can be refactored into this.
ResourcesDirectory.html
Curated resources directory for organizations, reading, and tools. Grouped sections with compact cards and tags such as Beginner friendly, In depth, and Urgent.
