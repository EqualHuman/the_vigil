This page gathers the high-level outline for the template library. Each section groups related files. Every row is a planned template set with a short description and a status tag. When a template moves from idea to in use, only this page needs to be touched to keep the track in view.

How to use

Quick notes
  • Change the status pill text and class when work begins, pauses, or ships.
  • Drop links or dates into the notes column once a template lives in the repo.
  • Duplicate full sections when new subprojects or collections appear.
