For educators

Assignable modules • discussion prompts • no paywall (for now)

LabContext is designed to be easy to assign: short modules, calm feedback, and clear learning objectives. Students don’t need accounts. Nothing is timed. Modules are built to work as pre-class prep or in-class discussion starters.

How to use LabContext

  • Pre-class: assign one module (6–10 min) + one lesson (8–12 min).
  • In-class: run the module together and pause for discussion at each decision point.
  • Assessment: ask students to write a short reflection on tradeoffs and responsibility.
