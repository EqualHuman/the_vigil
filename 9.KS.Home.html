FAQ

Quick answers before you book.

Do you host the website?

Yes — I can host it on GitHub Pages (free) or help connect a custom domain. You can also host elsewhere if you prefer.

Can guests RSVP on the site?

Absolutely. I typically connect a clean RSVP form (like Google Forms) so you can easily view responses and export a list.

What do you need from me?

Names, date, venue details, schedule, registry links, and any photos. I’ll send a simple checklist after booking.

Can I update it later?

Yes. You can book a small update add-on, or I can set it up so you can edit certain text easily.
